# Player app

Web based mp3 player.

## Run app

Single shoot (assumption: you are in root dir)

```
> cd client && yarn install && yarn build
> cd ../file_server && yarn install && yarn start
```

Visit & enjoy the app on `http://localhost:8083/index.html`
